Para mostrar una advertencia sin llamar a una función en una ventana de tkinter, puedes usar
tkinter.messagebox.showwarning() directamente, sin necesidad de una función de Python personalizada.
Simplemente, importa tkinter y messagebox y luego llama a showwarning pasando el título y el mensaje como argumentos,
como en messagebox.showwarning("Título", "Mensaje de advertencia"). Esta es la forma directa de mostrar un diálogo de
advertencia estándar de la interfaz gráfica.

import tkinter as tk
from tkinter import messagebox

# Crea la ventana principal, pero no la muestra en pantalla (opcional)
root = tk.Tk()
root.withdraw()

# Muestra directamente el mensaje de advertencia
messagebox.showwarning("Advertencia", "¡Esta es una advertencia de ejemplo!")

# La ventana principal de Tkinter se cerrará una vez que el cuadro de diálogo se cierre
